# 業務フローパターンの複数選択機能の実装

## 日時
2026年1月14日

## 実装内容

### 🎯 概要

出荷計画作成時に、複数の業務フローパターンを選択できるようにしました。1つの出荷の中で、商品によって異なる配送ルートを使用できるようになりました。

---

## 💡 背景・ニーズ

### ユースケース

**実際のビジネスシーン**:
- 同じ出荷計画の中で、複数の商品カテゴリがある
- 一部の商品はメーカーから直接フォワーダーに送る（直送）
- 一部の商品はメーカーから梱包業者を経由してフォワーダーに送る
- これらを1つの出荷計画としてまとめて管理したい

### 変更前の問題点

```
業務フローパターン *
┌────────────────────────────────────┐
│ 直送（メーカー → フォワーダー）    │▼  ← 1つしか選択できない
└────────────────────────────────────┘
```

**問題**:
- ❌ 1つの出荷計画で1つのフローしか選択できない
- ❌ 商品ごとに異なるフローを使用できない
- ❌ 複雑な出荷計画を作成できない

## ✨ 新機能

### 1. 複数選択

**機能**:
- ✅ チェックボックスで複数のフローパターンを選択
- ✅ 必要なパターンをすべて選択できる
- ✅ 選択数の表示（例: 選択中: 2件 / 4件）

---

### 2. 検索・フィルタリング

**機能**:
- ✅ パターン名で検索
- ✅ 説明文でも検索
- ✅ リアルタイムフィルタリング

**使用例**:
```

→ 「直送（メーカー → フォワーダー）」のみ表示

検索: "梱包業者"
```

**結果**:
- ✅ 1つの出荷計画で3つのフローを管理

## 📊 UI コンポーネントの詳細

### チェックボックスリスト

```tsx
<div className="border border-gray-300 rounded-lg max-h-64 overflow-y-auto">
  {flowPatternsData
    .filter(pattern => 
      pattern.name.toLowerCase().includes(flowPatternSearch.toLowerCase()) ||
      pattern.description.toLowerCase().includes(flowPatternSearch.toLowerCase())
    )
    .map(pattern => (
      <label className="flex items-start p-3 hover:bg-gray-50 cursor-pointer">
        <input
          type="checkbox"
          checked={formData.flowPatternIds.includes(pattern.id)}
          onChange={(e) => {
            if (e.target.checked) {
              // 追加
              setFormData({
                ...formData,
                flowPatternIds: [...formData.flowPatternIds, pattern.id]
              });
            } else {
              // 削除
              setFormData({
                ...formData,
                flowPatternIds: formData.flowPatternIds.filter(id => id !== pattern.id)
              });
            }
          }}
        />
        <div className="ml-3">
          <span className="font-medium">{pattern.name}</span>
          <span className="badge">{pattern.code}</span>
          {pattern.requiresPacking && <span className="badge-warning">梱包業者あり</span>}
          <p className="text-xs text-gray-600">{pattern.description}</p>
        </div>
      </label>
    ))
  }
</div>
```

**特徴**:
- ✅ スクロール可能（最大高さ: 256px）
- ✅ ホバー効果（背景色変化）
- ✅ リアルタイム検索フィルタ

---

### 選択済みタグ表示

```tsx
{formData.flowPatternIds.length > 0 && (
  <div className="mt-3 p-3 bg-blue-50 rounded-lg">
    <p className="text-xs font-medium text-gray-700 mb-2">
      選択されたフローパターン:
    </p>
    <div className="flex flex-wrap gap-2">

        const pattern = flowPatternsData.find(p => p.id === id);
        return (
          <span className="px-3 py-1 bg-white rounded-full border flex items-center">
            <span>{pattern.name}</span>
            <button
              onClick={() => setFormData({
                ...formData,
                flowPatternIds: formData.flowPatternIds.filter(pid => pid !== id)
              })}
              className="ml-2 text-gray-500 hover:text-red-600"
            >
              ×
            </button>
          </span>
        );
      })}
    </div>
  </div>
)}
```

**特徴**:
- ✅ 青い背景で選択内容を強調
- ✅ 各タグに削除ボタン
- ✅ フレックスレイアウトで複数行対応
