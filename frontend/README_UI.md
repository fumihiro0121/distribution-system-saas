# 物流管理システムSaaS - フロントエンドUI

## 概要
Next.js 14 + TypeScript + Tailwind CSSで構築された物流管理システムのUIデモ

## 実装済み画面

### 1. ログイン画面 (`/login`)
- メールアドレス・パスワード認証
- 役割別のデモアカウント
- レスポンシブデザイン

### 2. 管理者ダッシュボード (`/admin`)
- 統計カード（出荷予定、進行中、遅延、完了）
- 月別出荷実績グラフ
- 配送先別内訳
- 最近の出荷計画一覧
- ナビゲーションメニュー

### 3. フォワーダー画面 (`/forwarder`)
- 作業指示一覧（パレット積載、コンテナ積付）
- 入荷予定一覧（メーカー直送、梱包業者経由）
- 統計情報
- 作業開始・完了報告機能

### 4. 梱包業者画面 (`/packing`)
- 詳細な作業指示
  - 入荷予定商品情報
  - 商品別ラベル貼付指示
  - 梱包指示（段ボールサイズ、個数）
  - 段ボール貼付書類
  - 出荷先情報
- 作業完了報告
- 写真アップロード
- 作業指示書印刷

### 5. 配送先（スーパー）画面 (`/supermarket`) ⭐ 新機能
- 出荷概要（出荷日、到着予定日、トラッキング番号）
- 商品リスト
  - 商品名（英語・日本語）
  - JANコード、HSコード
  - 数量、単価、合計金額
  - サイズ、重量
- パレット構成
  - パレットコード、サイズ、重量
  - 積載段ボール一覧
  - 各段ボールの内容物
  - 3D積付図プレビュー（予定）
- CSV/PDF出力機能

## セットアップ

### 前提条件
- Node.js 18.x以上
- npm または yarn

### インストール

```bash
# 依存関係のインストール
npm install

# または
yarn install
```

### 開発サーバーの起動

```bash
npm run dev

# または
yarn dev
```

ブラウザで http://localhost:3000 を開きます

## 技術スタック

- **フレームワーク**: Next.js 14 (App Router)

- **ルーティング**: App Router (next/navigation)

## 配送先（スーパー）画面の特徴 ⭐

### 表示情報
1. **出荷概要**
   - 出荷コード、出荷名
   - 出荷元、フォワーダー
   - 出荷日、到着予定日
   - 総段ボール数、総パレット数
   - 総重量
   - トラッキング番号

2. **商品リスト（詳細）**
   - 商品名（英語・日本語表記）
   - JANコード（バーコード情報）
   - HSコード（関税分類番号）
   - 数量
   - 単価・合計金額（USD表記）
   - 商品サイズ（cm）
   - 重量（kg）

3. **パレット構成**
   - パレットコード
   - パレットサイズ（48"×40"など）
   - パレット高さ、重量
   - 積載段ボール数
   - 各段ボールの詳細
     - 段ボールコード
     - 内容物（商品名×個数）
     - 段ボール重量
   - 3D積付図プレビュー（実装予定）

4. **エクスポート機能**
   - 商品リストCSV出力
   - パレット構成PDF出力

### 使用シーン
- アメリカのスーパーが入荷予定の商品を事前確認
- 商品情報をバイヤー、倉庫担当者と共有
- 入荷準備（スペース確保、人員配置）
- 通関書類の事前確認
- 在庫管理システムへのデータ取り込み

## デザインシステム

### カラーパレット
- **プライマリ**: Indigo (管理者)
- **Blue**: フォワーダー
- **Green**: 梱包業者
- **Purple**: 配送先（スーパー）
- **成功**: Green
- **警告**: Yellow
- **エラー**: Red

### タイポグラフィ
- フォント: システムフォント（font-sans）
- 見出し: font-bold
- 本文: font-normal
- コード: font-mono

### コンポーネント
- カード: bg-white, rounded-lg, shadow
- ボタン: rounded-lg, hover効果
- テーブル: divide-y, hover:bg-gray-50
- バッジ: rounded-full, 色分け

## トラブルシューティング

### ログイン後に画面が表示されない
- ブラウザのlocalStorageをクリア
- ページをリロード

### スタイルが適用されない
```bash
# Tailwind CSSの再ビルド
npm run dev
```
