## 🎨 新機能

### 1. 商品マスタデータファイル

**ファイル**: `frontend/data/products.ts`

**データ構造**:
```typescript
export interface Product {
  id: number;
  inputDate: string;
  productName: string;
  costPrice: number;           // 仕入金額
  sku: string;
  janCode: string;
  asin: string;
  fnsku: string;
  hsCode: string;
  supplier: string;            // 仕入先
  unitWeight: string;          // 単品重さ
  setQuantity: string;         // セット個数
  setWeight: string;           // セット重さ
  setWeightPounds: string;     // セット重さ（パウンド）
  standardBoxQuantity: number; // 標準段ボールへの梱包数
  standardBoxSetQuantity: number; // 標準段ボールへの梱包セット数
  boxLengthCm: number;         // 段ボール縦（センチ）
  boxWidthCm: number;          // 段ボール横（センチ）
  boxHeightCm: number;         // 段ボール高さ（センチ）
  boxLengthInch: number;       // 段ボール縦（インチ）
  boxWidthInch: number;        // 段ボール横（インチ）
  boxHeightInch: number;       // 段ボール高さ（インチ）
  category?: string;           // カテゴリ
  notes?: string;              // 備考
}
```

**特徴**:
- ✅ 25件の商品データを登録
- ✅ カテゴリ別に整理
- ✅ Amazon情報（SKU、ASIN、FNSKU）
- ✅ 重量情報（kg、ポンド）
- ✅ 段ボール情報（cm、インチ）

#### B. フォームデータの自動保存

**機能**:
```typescript
// フォームデータが変更されたら自動でlocalStorageに保存
useEffect(() => {
  if (user) {
    localStorage.setItem('shipmentFormData', JSON.stringify(formData));
  }
}, [formData, user]);
```

**保存される情報**:
- ✅ 出荷日
- ✅ 出荷名

- ✅ 選択した配送先
- ✅ 配送先情報（国、港、住所）
- ✅ 納期情報
- ✅ 備考

**自動保存のタイミング**:
- ✅ 入力フィールドを変更した瞬間
- ✅ チェックボックスを選択した瞬間
- ✅ ドロップダウンで選択した瞬間

#### C. データの自動復元

**機能**:
```typescript
// ページ読み込み時に保存されたデータを復元
useEffect(() => {
  const savedFormData = localStorage.getItem('shipmentFormData');
  if (savedFormData) {
    setFormData(JSON.parse(savedFormData));
  }
}, []);
```

**復元される情報**:
- ✅ すべての入力内容
- ✅ すべての選択内容
- ✅ 配送先情報
