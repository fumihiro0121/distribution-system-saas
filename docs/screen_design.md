# 画面設計書

## 画面遷移図

```
[ログイン画面]
    ↓ 認証成功
[ダッシュボード]
    ├─→ [出荷計画一覧] (メーカー・管理者)
    │       ├─→ [出荷計画登録]
    │       └─→ [出荷計画詳細]
    │               ├─→ [段ボール一覧]
    │               │       ├─→ [段ボール登録・編集]
    │               │       └─→ [段ボール詳細]
    │               ├─→ [パレット一覧]
    │               │       ├─→ [パレット登録・編集]
    │               │       └─→ [パレット詳細]
    │               └─→ [コンテナ一覧]
    │                       ├─→ [コンテナ登録・編集]
    │                       └─→ [コンテナ詳細]
    │
    ├─→ [作業指示一覧] (梱包業者・フォワーダー)
    │       └─→ [作業指示詳細]
    │               ├─→ [作業報告]
    │               └─→ [入荷確認]
    │
    ├─→ [トラッキング画面] (全ユーザー)
    │       └─→ [履歴詳細]
    │
    ├─→ [マスタ管理] (管理者)
    │       ├─→ [商品マスタ]
    │       ├─→ [取引先マスタ]
    │       ├─→ [サイズマスタ]
    │       └─→ [書類・ラベルテンプレート]
    │
    ├─→ [ユーザー管理] (管理者)
    │       └─→ [ユーザー登録・編集]
    │
    ├─→ [レポート] (管理者)
    │       ├─→ [出荷実績レポート]
    │       ├─→ [遅延分析レポート]
    │       └─→ [コスト分析レポート]
    │
    └─→ [設定] (全ユーザー)
            ├─→ [プロフィール編集]
            └─→ [通知設定]
```

---

## 画面詳細仕様

### 1. ログイン画面

**URL**: `/login`

**表示項目**:
- ロゴ
- メールアドレス入力フィールド
- パスワード入力フィールド
- ログインボタン
- パスワードを忘れた場合のリンク

**機能**:
- 認証処理
- セッション管理
- エラーメッセージ表示

---

### 2. ダッシュボード

**URL**: `/dashboard`

**表示項目**（役割によって異なる）:

#### 2.1 メーカー向け
- 出荷予定件数（今月）
- 進行中の出荷件数
- 遅延している出荷件数
- 完了した出荷件数（今月）
- 最近の出荷計画一覧（5件）
- グラフ（月別出荷実績）

#### 2.2 梱包業者向け
- 未完了の作業指示件数
- 今日の作業予定件数
- 入荷予定件数（今週）
- 完了した作業件数（今月）
- 作業指示一覧（期限順）

#### 2.3 フォワーダー向け
- 入荷予定件数（今週）
- 積載待ちパレット数
- 出荷予定コンテナ数（今月）
- 完了した出荷件数（今月）
- 入荷予定一覧
- コンテナ出荷予定カレンダー

#### 2.4 管理者向け
- 全体の統計情報
- システム全体のアクティビティ
- 各組織の進捗状況
- アラート・警告一覧

**機能**:
- グラフの期間切替（今日、今週、今月、今年）
- クイックアクセスボタン
- 通知の確認

---

### 3. 出荷計画一覧

**URL**: `/shipments`

**表示項目**:
- 検索・フィルタエリア
  - 出荷コード
  - 出荷名
  - メーカー名
  - フォワーダー名
  - ステータス
  - 納期範囲
- 出荷計画リスト（テーブル）
  - 出荷コード
  - 出荷名
  - メーカー名
  - フォワーダー名
  - 納期
  - ステータス
  - 段ボール数
  - パレット数
  - コンテナ数
  - 進捗率
  - アクションボタン（詳細、編集、削除）
- ページネーション
- 新規作成ボタン

**機能**:
- 検索・フィルタリング
- ソート（各列でソート可能）
- CSV/Excelエクスポート
- 新規出荷計画作成
- 編集・削除
- 詳細表示

---

### 4. 出荷計画登録・編集

**URL**: `/shipments/new` または `/shipments/:id/edit`

**入力項目**:

#### 基本情報
- 出荷名
- メーカー選択
- 業務フローパターン選択（直送 or 梱包業者経由）
- 梱包業者選択（梱包業者経由の場合）
- フォワーダー選択
- 納期

#### 配送先情報
- 仕向地国
- 仕向地港
- 仕向地住所

#### 備考
- 備考欄

**機能**:
- 入力バリデーション
- 保存ボタン
- キャンセルボタン
- 保存後、出荷計画詳細画面へ遷移

### 6. 段ボール登録・編集

**URL**: `/shipments/:shipmentId/cartons/new` または `/cartons/:id/edit`

**入力項目**:

#### 段ボール情報
- 段ボールサイズ選択（プリセット or カスタム）
- カスタムサイズ（縦×横×高さ）
- 重量
- 配送先住所

#### 内容物
- 商品選択 + 数量（複数行追加可能）
  - 商品名
  - 数量

#### 貼付ラベル
- 必要ラベルチェックボックス
  - FNSKUラベル
  - 栄養成分シール
  - その他

#### 貼付書類
- 必要書類チェックボックス
  - 送り状
  - Amazon FBA送り状
  - FDA事前通知書
  - FSVP代理人同意書
  - その他

#### パレット割当
- パレット選択（オプション）

**機能**:
- 商品の追加・削除
- バーコード/QRコード自動生成
- ラベル・書類のプレビュー
- 保存ボタン
- キャンセルボタン

---

### 7. パレット登録・編集

**URL**: `/shipments/:shipmentId/pallets/new` または `/pallets/:id/edit`

**入力項目**:

#### パレット情報
- パレットサイズ選択
- 配送先住所

#### 積載段ボール
- 段ボール選択（複数選択可能）
- 段数指定
- 積付位置指定（ビジュアルエディタ）

**表示情報（自動計算）**:
- 総重量
- 総容積
- 箱数

**機能**:
- 段ボールの追加・削除
- 積付図のビジュアル表示
- 積付シミュレーション
- バーコード/QRコード自動生成
- 保存ボタン
- キャンセルボタン

---

### 13. マスタ管理

#### 13.1 商品マスタ

**URL**: `/master/products`

**表示項目**:
- 商品一覧テーブル
  - 商品コード
  - 商品名
  - メーカー名
  - サイズ（縦×横×高さ）
  - 重量
  - HSコード
  - 原産国
  - アクションボタン
- 新規登録ボタン

**登録・編集項目**:
- 商品コード（内部管理用）
- 商品名（日本語・英語）
- メーカー選択
- **識別コード**
  - JANコード
  - HSコード（HTSコード）
  - 原産国
- **価格情報**
  - 原価
  - 売価
  - 通貨選択（JPY、USD等）
- **Amazon情報**
  - ASIN
  - Seller SKU
  - FNSKU
- **サイズ・重量**
  - 長さ（cm）
  - 幅（cm）
  - 高さ（cm）
  - 重量（kg）
  - 容積重量（自動計算・表示のみ）
- **セット商品設定**
  - セット商品フラグ（チェックボックス）
  - 基本商品選択（セット商品の場合）
  - セット数（2個セット、3個セットなど）
  - ※セット商品は独立した商品として全情報を設定可能
- デフォルトラベル（チェックボックス）
- デフォルト書類（チェックボックス）
- 説明
