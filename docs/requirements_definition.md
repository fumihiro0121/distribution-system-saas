# 物流管理システムSaaS 要件定義書

## 1. システム概要

### 1.1 目的
メーカーからの商品を国際輸出するまでの物流プロセスを、複数のステークホルダー間で一元管理し、トレーサビリティを確保するシステム

### 1.2 対象ユーザー
- メーカー（商品出荷元）
- 国内梱包業者（例：株式会社福富運送）
- フォワーダー（例：佐川、日本ジャパントラスト）
- 配送先（Amazon FBA、アメリカのスーパー等）⭐ 新規追加
- システム管理者

---

## 2. 業務フロー

### 2.1 パターンA：メーカー → フォワーダー → 輸出
```
メーカー → フォワーダー → 輸出
         （書類・ラベル貼付）
         （パレット積付）
         （コンテナ積付）
```

### 2.2 パターンB：メーカー → 梱包業者 → フォワーダー → 輸出
```
メーカー → 梱包業者 → フォワーダー → 輸出
         （ラベル貼付）  （書類貼付）
         （詰替え）      （パレット積付）
                        （コンテナ積付）
```

---

## 3. 機能要件

### 3.1 出荷管理機能

#### 3.1.1 出荷計画登録
- **目的**: メーカーまたは管理者が出荷計画を登録
- **入力項目**:
  - 出荷ID（自動採番）
  - 商品情報（商品名、商品コード、数量）
  - 出荷元（メーカー情報）
  - 出荷先（梱包業者 or フォワーダー）
  - 最終仕向地（輸出先国・住所）
  - 納期
  - 業務フローパターン（A or B）
- **機能**:
  - 出荷計画の新規作成
  - 出荷計画の編集・削除
  - 出荷計画の複製

#### 3.1.2 段ボール情報管理
- **目的**: 商品を梱包する段ボールの詳細管理
- **管理項目**:
  - 段ボールID（自動採番 + バーコード/QRコード生成）
  - 段ボールサイズ（縦×横×高さ、cm）
  - 重量（kg）
  - 内容物（商品名、数量のリスト）
  - 貼付必要書類（送り状、Amazon FBA送り状、FDA事前通知書、FSVP代理人同意書など）
  - 貼付ラベル（FNSKUラベル、栄養成分シールなど）
  - 配送先情報
  - 所属パレットID
  - ステータス（梱包前、梱包済、出荷済、配達済）
- **機能**:
  - 段ボール情報の登録・編集
  - バーコード/QRコードの印刷
  - 段ボール単位でのステータス更新
  - 段ボールの検索・フィルタリング

#### 3.1.3 パレット情報管理
- **目的**: 段ボールを積載するパレットの管理
- **管理項目**:
  - パレットID（自動採番 + バーコード/QRコード生成）
  - パレットサイズ（標準、L、LL など）
  - 積載段ボールリスト（段ボールID、段数、箱数）
  - 総重量（kg）
  - 総容積（m³）
  - 配送先情報
  - 所属コンテナID
  - ステータス（積載前、積載中、積載完了、出荷済、配達済）
- **機能**:
  - パレット情報の登録・編集
  - 段ボールのパレットへの割当
  - パレット積付図の表示
  - バーコード/QRコードの印刷
  - パレット単位でのステータス更新

#### 3.1.4 コンテナ情報管理
- **目的**: パレットを積載するコンテナの管理
- **管理項目**:
  - コンテナID
  - コンテナサイズ（20ft、40ft など）
  - 積載パレットリスト
  - 最大積載量
  - 輸出先（国、港、住所）
  - 船名・便名
  - 出港日
  - 到着予定日
  - ステータス（積載前、積載完了、出港済、到着済）
- **機能**:
  - コンテナ情報の登録・編集
  - パレットのコンテナへの割当
  - コンテナ積付図の表示
  - 積載率の表示

### 3.2 作業指示機能

#### 3.2.1 梱包業者向け作業指示
- **表示内容**:
  - 入荷予定商品リスト（商品名、数量、入荷元段ボールサイズ・数）
  - 貼付ラベル指示（商品ごとのFNSKUラベル、栄養成分シールなど）
  - 詰替え指示（新段ボールサイズ、商品名、梱包数）
  - 貼付書類指示（段ボールごとの送り状、FBA送り状、FDA書類など）
  - 出荷先フォワーダー情報
  - 納期
- **機能**:
  - 作業指示書の表示・印刷
  - 作業完了報告（段ボールID入力、写真アップロード）

#### 3.2.2 フォワーダー向け作業指示
- **表示内容**:
  - 入荷予定商品リスト
    - メーカーからの直送分（商品名、段ボールサイズ、数量、パレット情報）
    - 梱包業者からの入荷分（商品名、段ボールサイズ、数量）
  - パレット積付指示（段ボールID、パレットID、段数、箱数）
  - 貼付書類指示（パレット・段ボールごと）
  - コンテナ積付指示（パレットID、コンテナID）
  - 輸出先情報（国、港、住所）
  - 納期・船便情報
- **機能**:
  - 作業指示書の表示・印刷
  - 入荷確認（バーコードスキャン）
  - 作業完了報告（パレットID、コンテナID入力、写真アップロード）

### 3.3 進捗管理・トラッキング機能

#### 3.3.1 リアルタイム進捗表示
- **表示内容**:
  - 出荷計画全体の進捗率
  - 各段ボールのステータス
  - 各パレットのステータス
  - 各コンテナのステータス
- **機能**:
  - ダッシュボード表示
  - ステータス別フィルタリング
  - 遅延アラート表示

#### 3.3.2 トレーサビリティ
- **機能**:
  - 商品単位での追跡（どの段ボール→どのパレット→どのコンテナ）
  - 段ボール単位での追跡（どこからどこへ、誰が作業）
  - パレット単位での追跡
  - 履歴の表示（日時、作業者、場所、写真）

### 3.4 書類・ラベル管理機能

#### 3.4.1 書類テンプレート管理
- **管理対象**:
  - 送り状
  - Amazon FBA送り状
  - FDA事前通知書
  - FSVP代理人同意書
  - インボイス
  - パッキングリスト
  - その他
- **機能**:
  - テンプレートの登録・編集
  - 変数の差込（商品名、数量、住所など）
  - PDF生成
  - 印刷

#### 3.4.2 ラベル管理
- **管理対象**:
  - FNSKUラベル
  - 栄養成分シール
  - 商品ラベル
  - 段ボールラベル
  - パレットラベル
- **機能**:
  - ラベルデザインの登録・編集
  - バーコード/QRコード生成
  - 一括印刷

### 3.5 マスタ管理機能

#### 3.5.1 商品マスタ
- 商品コード（内部管理用）
- 商品名（日本語・英語）
- JANコード（JAN-13、JAN-8）
- HSコード（HTSコード）- 輸出入品目分類番号
- 原産国
- メーカー
- **価格情報**
  - 原価（仕入価格）
  - 売価（販売価格）
  - 通貨設定
- **Amazon情報**
  - ASIN（Amazon Standard Identification Number）
  - SKU（Stock Keeping Unit）
  - FNSKU（Fulfillment Network SKU）
- **サイズ・重量情報**
  - 長さ（cm）
  - 幅（cm）
  - 高さ（cm）
  - 重量（kg）
  - 容積重量（自動計算）
- **セット商品管理**
  - 基本商品ID（1個単位の商品）
  - セット数（1個、2個セット、3個セットなど）
  - セット商品として、JANコード、原価、売価、ASIN、SKU、FNSKU、サイズ、重量が個別に設定可能
- 必要ラベル（デフォルト）
- 必要書類（デフォルト）
- 備考

#### 3.5.2 取引先マスタ
- 取引先コード
- 取引先名
- 種別（メーカー、梱包業者、フォワーダー）
- 住所
- 電話番号
- 担当者情報
- メールアドレス

#### 3.5.3 段ボールサイズマスタ
- サイズコード
- サイズ名（S、M、L など）
- 寸法（縦×横×高さ）
- 最大積載重量

#### 3.5.4 パレットサイズマスタ
- サイズコード
- サイズ名
- 寸法
- 最大積載重量

#### 3.5.5 コンテナサイズマスタ
- サイズコード
- サイズ名（20ft、40ft など）
- 内寸
- 最大積載重量
- 最大容積

### 3.6 ユーザー管理・権限管理機能

#### 3.6.1 ユーザー管理
- ユーザーID
- ユーザー名
- 所属組織
- 役割（管理者、メーカー、梱包業者、フォワーダー、配送先）⭐ 配送先追加
- メールアドレス
- パスワード

#### 3.6.2 権限管理
- **管理者**: 全機能アクセス可
- **メーカー**: 出荷計画登録、進捗閲覧
- **梱包業者**: 作業指示閲覧、作業報告、進捗更新
- **フォワーダー**: 作業指示閲覧、作業報告、進捗更新、コンテナ管理
- **配送先（スーパー等）**: 自分宛の出荷情報閲覧（商品、段ボール、パレット構成）⭐ 新規追加

### 3.7 配送先（スーパー等）向け閲覧機能 ⭐ 新規追加

#### 3.7.1 目的
配送先（アメリカのスーパー等）が、自分宛の出荷情報を閲覧できるようにし、入荷準備を効率化

#### 3.7.2 閲覧可能情報
- **出荷概要**
  - 出荷コード、出荷名
  - 出荷元（メーカー名）
  - 出荷予定日、到着予定日
  - 総段ボール数、総パレット数
  - 総重量、総容積

- **商品リスト**
  - 商品名（英語）
  - 商品コード、JANコード
  - 数量
  - 単価、合計金額
  - サイズ、重量

- **段ボール構成**
  - 段ボールコード
  - 段ボールサイズ（寸法、重量）
  - 内容物（どの商品が何個入っているか）
  - 所属パレット

- **パレット構成**
  - パレットコード
  - パレットサイズ（寸法、重量）
  - 積載段ボール数
  - 段ボールリスト（各段ボールのコード、内容物）
  - 積付図（3D可視化またはイメージ）

- **書類**
  - インボイス
  - パッキングリスト
  - 原産地証明書
  - その他関連書類のダウンロード

- **トラッキング情報**
  - 現在のステータス
  - 出荷日時
  - 到着予定日時
  - 配送業者、トラッキング番号

#### 3.7.3 機能
- **検索・フィルタ**
  - 出荷日範囲
  - ステータス
  - 商品名

- **エクスポート**
  - 商品リストCSV出力
  - パレット構成PDF出力
  - 全情報Excel出力

- **通知設定**
  - 出荷時の通知
  - 到着予定日の通知（3日前、1日前）

#### 3.7.4 アクセス制御
- 配送先ユーザーは自分宛の出荷のみ閲覧可能
- 編集権限なし（閲覧・ダウンロードのみ）
- ログイン必要

### 3.8 通知機能

#### 3.8.1 通知タイミング
- 出荷計画が登録されたとき
- 作業指示が発行されたとき
- 商品が入荷したとき
- 作業が完了したとき
- 出荷したとき
- 納期が近づいたとき（3日前、1日前）
- 遅延が発生したとき
- 配送先向け：出荷されたとき、到着予定日が近づいたとき ⭐ 新規追加

#### 3.8.2 通知方法
- メール通知
- システム内通知

### 3.8 輸出計算・書類生成機能

#### 3.8.1 輸出金額・重量計算
- **商品別計算**
  - 各商品の個数 × 単価
  - 各商品の個数 × 重量
  - 通貨換算（JPY → USD等）
- **出荷単位計算**
  - 段ボール別の合計金額・重量
  - パレット別の合計金額・重量
  - コンテナ別の合計金額・重量
  - 出荷全体の総額・総重量
- **配送業者向け情報生成**
  - FedEx商業送り状用データ
  - DHL用データ
  - UPS用データ
  - インボイス金額計算
- **履歴保存と再利用**
  - 過去の輸出データを保存
  - テンプレートとして再利用
  - 商品情報の自動入力

#### 3.8.2 AI最適化提案機能
- **段ボールサイズ提案**
  - 商品サイズ・数量から最適な段ボールサイズを提案
  - Amazon FBA規定内のサイズを優先提案
  - 一般輸出向けサイズも選択可能
  - 複数の選択肢を提示（最適、代替案1、代替案2）
- **梱包方法提案**
  - 商品の組み合わせ提案（どの商品をどの段ボールに）
  - 段ボール内の配置提案
  - 緩衝材の必要量提案
- **パレット積付提案**
  - 段ボールの最適配置パターン提案
  - 重量バランスを考慮
  - 高さ制限を考慮（Amazon FBA、一般輸出）
  - 段数・配置の3D可視化
- **管理者による修正機能**
  - AI提案を基に、商品個数、段ボール個数、サイズを手動調整
  - 調整後の積載率、重量バランスを再計算
  - 修正履歴の保存

#### 3.8.3 配送先別規定管理
- **Amazon FBA倉庫向け**
  - 段ボールサイズ制限
    - 標準サイズ: 25インチ × 任意の寸法、重量50ポンド以下
    - 大型サイズ: 25インチ超、重量50ポンド超
  - パレットサイズ制限
    - 標準パレット: 48インチ × 40インチ、高さ72インチ以下
    - 重量: 1,500ポンド以下
  - ラベル要件: FNSKUラベル、配送先ラベル
- **Amazon AWD倉庫向け**
  - パレットサイズ制限
  - 重量制限
  - ラベル要件
- **一般輸出（スーパー等）向け**
  - 標準パレット: 48インチ × 40インチ（1219mm × 1016mm）
  - 高さ制限: 72-96インチ（1829-2438mm）が一般的
  - 20ftコンテナ: パレット10-11枚
  - 40ftコンテナ: パレット20-24枚
- **規定チェック機能**
  - 入力時に自動チェック
  - 規定超過時に警告表示
  - 代替案の自動提示

### 3.9 作業指示の詳細化

#### 3.9.1 梱包業者向け詳細作業指示
- **入荷情報**
  - メーカーからの商品リスト（商品名、JANコード、数量）
  - 入荷予定日時
  - 段ボール個数、パレット枚数
- **ラベル貼付作業**
  - 商品別のラベル一覧
    - FNSKUラベル（商品ごと）
    - 栄養成分ラベル
    - Made in Japan等のラベル
  - ラベルの貼付位置指示
  - ラベルの印刷機能
- **梱包作業**
  - どの商品を何個ずつ、どのサイズの段ボールに梱包するか
  - 段ボールの組立方法
  - 緩衝材の使用方法
- **段ボール貼付作業**
  - 段ボールごとに貼付する書類・ラベル
    - 配送ラベル
    - Amazon FBA送り状
    - FDA事前通知書
    - FSVP同意書
  - 貼付位置の図解
- **パレット積載作業**
  - どの段ボールをどのパレットに何箱積むか
  - 積み方の図解（段数、配置）
  - パレット包装方法（ストレッチフィルム等）
- **出荷準備**
  - フォワーダーへの出荷日時
  - 配送先住所
  - 連絡事項

#### 3.9.2 フォワーダー向け詳細作業指示
- **入荷情報**
  - メーカーからの直送分
    - 商品リスト、段ボール個数、パレット枚数
    - 入荷予定日時
  - 梱包業者からの入荷分
    - 商品リスト、段ボール個数、パレット枚数
    - 入荷予定日時
- **検品作業**
  - 入荷確認リスト
  - バーコードスキャン機能
  - 破損チェック
- **パレット積載作業**
  - 段ボールからパレットへの積載指示
  - 段数、配置の図解
  - 配送先別の仕分け
- **コンテナ積載作業**
  - パレットからコンテナへの積載指示
  - 積載順序
  - 固定方法
- **輸出手続き**
  - 必要書類リスト
  - インボイス金額
  - 総重量
  - HSコード一覧
  - 配送業者（FedEx、DHL等）の送り状データ
- **出荷情報**
  - 船便情報（船名、便名、出港日）
  - 航空便情報（便名、出発日時）
  - トラッキング番号

## 4. 非機能要件

### 4.1 パフォーマンス
- ページ読込時間: 3秒以内
- 同時接続ユーザー数: 100人以上
- データ保持期間: 3年以上

### 4.2 セキュリティ
- SSL/TLS通信
- ユーザー認証（パスワード + 多要素認証オプション）
- ロールベースアクセス制御
- 操作ログの記録
- データバックアップ（日次）

### 4.3 可用性
- 稼働率: 99.5%以上
- 定期メンテナンス: 月1回（事前通知）

### 4.4 互換性
- ブラウザ: Chrome、Edge、Safari（最新2バージョン）
- デバイス: PC、タブレット対応
- モバイル対応: レスポンシブデザイン

### 4.5 拡張性
- マルチテナント対応
- API提供（外部システム連携用）
- データエクスポート機能（CSV、Excel）

## 5. システム構成案

### 5.1 技術スタック（例）
- **フロントエンド**: React / Next.js
- **バックエンド**: Node.js (Express) / Python (FastAPI)
- **データベース**: PostgreSQL
- **ファイルストレージ**: AWS S3 / Azure Blob Storage
