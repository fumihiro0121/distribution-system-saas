# 物流管理システムSaaS

メーカー、梱包業者、フォワーダー間の国際物流プロセスを一元管理するSaaSプラットフォーム

## 📋 概要

本システムは、メーカーからの商品が国際輸出されるまでの複雑な物流プロセスを、複数のステークホルダー間でリアルタイムに共有・管理できるクラウドベースのシステムです。

### 主な特徴

- 🚚 **エンドツーエンドの物流管理**: メーカーから輸出先まで一貫した管理
- 📦 **段ボール・パレット・コンテナの階層管理**: 商品の梱包状態を詳細に追跡
- 👥 **マルチテナント対応**: 複数の組織が同一プラットフォームで協業
- 🔍 **完全なトレーサビリティ**: 商品がどこにあるか常に把握可能
- 📄 **書類・ラベル管理**: FDA書類、FBAラベルなど各種書類を自動生成
- 📊 **リアルタイムダッシュボード**: 進捗状況を視覚的に把握
- 🔐 **ロールベースアクセス制御**: 組織・役割に応じた権限管理
- 💰 **輸出計算機能**: 商品別・段ボール別の金額・重量を自動計算、配送業者向けフォーマット生成
- 🤖 **AI最適化提案**: 段ボールサイズ、パレット積付を自動提案、3D可視化
- 📋 **テンプレート機能**: 頻繁な出荷パターンを保存してワンクリック作成
- ✅ **規定自動チェック**: Amazon FBA/一般輸出の規定を自動チェック、警告表示
- 🔄 **セット商品管理**: 同じ商品の2個セット、3個セットを個別に管理

## 🎯 対象ユーザー

1. **メーカー**: 商品の出荷計画を管理
2. **梱包業者**: 商品の梱包作業を管理（ラベル貼付、詰替え等）
3. **フォワーダー**: パレット・コンテナへの積載、輸出業務を管理
4. **システム管理者**: マスタデータ、ユーザー、レポートを管理

## 📦 主要エンティティ

```
出荷計画 (Shipments)
    ├── 段ボール (Cartons)
    │   └── 商品 (Products)
    ├── パレット (Pallets)
    │   └── 段ボール (Cartons)
    └── コンテナ (Containers)
        └── パレット (Pallets)
```

### データの流れ
1. **メーカー**が出荷計画を作成
2. 商品を**段ボール**に梱包
3. 段ボールを**パレット**に積載
4. パレットを**コンテナ**に積載
5. コンテナを輸出

各ステップで、適切なステークホルダーに**作業指示**が自動発行され、完了報告により進捗が更新されます。

## 🌟 プロジェクトのビジョン

物流業界のDXを推進し、国際貿易におけるサプライチェーンの透明性と効率性を大幅に向上させることで、グローバルビジネスの発展に貢献します。

**私たちのミッション**:
- 物流プロセスの見える化による業務効率の向上
- ステークホルダー間のシームレスな情報共有
- トレーサビリティの確保によるコンプライアンス強化
- データドリブンな意思決定の支援

このシステムが、物流に関わる全ての方々の業務を楽にし、よりスムーズな国際物流を実現できることを願っています。
