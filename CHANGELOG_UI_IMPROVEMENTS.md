### 2. 出荷名の形式改善 ✅

#### 新しい形式
```
YYYY-MM-DD 配送先名＋配送先タイプ向け輸出
```

#### 例
- `2026-01-15 米国マルカイ＋米国Amazon-AWD向け輸出`
- `2026-01-18 米国Walmart＋スーパーマーケット向け輸出`
- `2026-01-20 米国Amazon-FBA向け輸出`

#### 利点
- 日付で区別しやすい
- 配送先が一目で分かる
- 配送先タイプが明確

   - 「出荷計画を作成」をクリック
